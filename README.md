# Parallel Programming

Various parallel programming implementations.

- Cardiac Simulation
- Sparse Matrix-Vector Multiplication
- Jacobi Stencil
- Game of Life
- Noise Remover
- N-Queens Problem
- Marching Squares

Also includes header files for:

- Safe CUDA calls
- Read MatrixMarket matrices in COO format
- COO to/from CSR
- COO to/from CSC

Use Rust, C and CUDA!
